**df [опции]...[файл]**<br>
*df v9.1*

Просмотр использования дискового пространства файловых систем.

| Опция | Действие                                                                         |
| :---- | :------------------------------------------------------------------------------- |
| `-a, --all`        | показывает все файловые системы, включая псевдо (`tmpfs`, `devtmpfs`), дублирующиеся и недоступные    |
| `-B, --block-size=SIZE`   | масштабирует размеры на заданный размер блока. Пример: -BM выводит в мегабайтах (1 MB = 1048576 байт) |
| `-h, --human-readable`    | показывает размеры в удобных единицах (КБ, МБ, ГБ) в степенях 1024                                    |
| `-H, --si`                | работает как `-h`, но размеры в степенях 1000 (например, 1.1G)                                        |
| `-i, --inodes`            | показывать использование inode вместо объёма блоков. Полезно при большом количестве маленьких файлов  |
| `-k`                      | работает как `--block-size=1K` — размеры в килобайтах                                                 |
| `-l, --local`             | ограничивает вывод только локальными файловыми системами, без сетевых                                 |
| `--no-sync`               | не вызывает sync перед получением информации (по умолчанию)                                           |
| `--output[=FIELD_LIST]`   | кастомизирует вывод по полям. Если `FIELD_LIST` не задан, выводятся все поля                          |
| `-P, --portability`       | вывод в POSIX-формате (таблица всегда 6 колонок)                                                      |
| `--sync`                  | вызывает `sync` перед получением информации, чтобы данные были точными                                |
| `--total`                 | выводит суммарное использование всех разделов                                                         |
| `-t, --type=TYPE`         | показывать только файловые системы указанного типа (`ext4`, `ntfs`, `vfat` и т.д.)                    |
| `-T, --print-type`        | показывает тип файловой системы                                                                       |
| `-x, --exclude-type=TYPE` | исключает файловые системы определённого типа                                                         |
| `-v`                      | игнорируется, оставлено для совместимости                                                             |
| `--help`                  | показывает справку по команде и выйти                                                                 |
| `--version`               | показывает версию утилиты `df`                                                                        |

**Единицы отображения (`SIZE`)**
- Значения отображаются в **единицах, указанных через `--block-size`**, или через переменные окружения `DF_BLOCK_SIZE`, `BLOCK_SIZE` или `BLOCKSIZE`.
- Если единицы не заданы, по умолчанию используется **1024 байта** (или 512 байт, если установлена переменная `POSIXLY_CORRECT`).
  
**Формат SIZE:**
- Целое число с необязательной единицей, например: `10K` = 10 × 1024 байт.
- Доступные единицы (степени **1024**): `K`, `M`, `G`, `T`, `P`, `E`, `Z`, `Y`, `R`, `Q`
- Доступные единицы (степени **1000**): `KB`, `MB`, `GB`, ...
- Можно использовать бинарные префиксы: `KiB = K`, `MiB = M` и так далее.
  
**Список полей для `--output` (`FIELD_LIST`)**
- Поля перечисляются через запятую.
- Доступные имена полей:

| Поле     | Описание                                         |
| -------- | ------------------------------------------------ |
| `source` | Устройство или раздел (например, `/dev/sda1`)    |
| `fstype` | Тип файловой системы (`ext4, ntfs, vfat` и т.д.) |
| `itotal` | Общее количество inode                           |
| `iused`  | Использованные inode                             |
| `iavail` | Свободные inode                                  |
| `ipcent` | Процент использования inode                      |
| `size`   | Общий размер файловой системы                    |
| `used`   | Используемое пространство                        |
| `avail`  | Доступное пространство                           |
| `pcent`  | Процент использования                            |
| `file`   | Проверяемый файл (точка монтирования)            |
| `target` | Точка монтирования                               |
