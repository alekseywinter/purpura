
**get-oui [опции]**

Получает обновление реестра MAC-производителей с сервера IEEE для дальнейшего использования в arp-scan. 

| Опция | Действие                                                                         |
| :---- | :------------------------------------------------------------------------------- |
| -f    | Указывает место для сохранения. По умолчанию: `/usr/share/arp-scan/ieee-oui.txt` |
| -v    | Показывает подробный прогресс обновления                                         |
| -h    | Показывает данную справку                                                |
