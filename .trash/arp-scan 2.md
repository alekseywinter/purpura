# ARP  
  
## RFC  
[826](https://www.rfc-editor.org/rfc/rfc826)  
  
## Как работает  
- Преобразует IP в MAC  
- Broadcast запросы в локальной сети  
  
## Формат пакета  
- H[[ethernet-frame]]pe, Protocol type  
- Hardware size, Protocol size  
- Opcode, Sender MAC/IP, Target MAC/IP
